Visibility Information

a) Toolbox

If the first lock is open, the entity can be found and added by another project in the toolbox
(via the " + Add" menu under Entities/Sources/Digitals).
If the first lock is closed, the entity is invisible to other projects and cannot be added to them.

b) Public

If the second lock is open, the entity has a URI and a website on https://www.geovistory.org.
Its properties are published as RDF on the public community SPARQL endpoint (at https://www.geovistory.org/sparql).
If the second lock is closed, the entity has no URI and no website.
Its properties are not visible in the public community SPARQL endpoint.
