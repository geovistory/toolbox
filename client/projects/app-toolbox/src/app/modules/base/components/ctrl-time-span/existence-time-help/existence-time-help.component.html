If you are in a hurry

  • For one date, use «At some time within»
  • For begin and end dates, use «Begin (latest possible)» and «End (earliest possible)»

If you want to understand

All events and phenomena have a certain duration. Geovistory provides different properties to narrow down this duration:

Please note that each calendar date has a certain duration. For example:

  • «1st of June 1981» has a duration of one day
  • «1981» has a duration of one year

Geovistory supports different levels of durations: years, months, days, hours, minutes and seconds.

These durations can be used to narrow down the duration of phenomena. This happens when you use «At some time within» and «Ongoing throughout».

At some time within

Use «At some time within», if the phenomenon happened at some time within the duration of the date you enter.

Example: The person was born at some time within the day 1st of June 1864.

Ongoing throughout

Use «ongoing throughout» if the phenomenon was ongoing throughout the whole duration of the date you enter.

Example: The exercise of the consul's office was ongoing throughout the year 54 BC.

Begin (earliest possible)

«Begin (earliest possible)» specifies the earliest possible moment the phenomenon could have begun to exist.

Use this option, if the phenomenon may have existed during and after, but surely not before the date you enter.

Begin (latest possible)

«Begin (latest possible)» specifies the latest possible moment the phenomenon could have begun to exist.

Use this option, if the phenomenon surely existed after the date you enter.
If you are not sure wheter to use «Begin (earliest possible)» or «Begin (latest possible)», use «Begin (latest possible)».

End (earliest possible)

«End (earliest possible)» specifies the earliest possible moment the phenomenon could have ended to exist.

Use this option, if the phenomenon surely existed before this moment.
If you are not sure wheter to use «End (latest possible)» or «End (earliest possible)», use «End (earliest possible)»

End (latest possible)

«End (latest possible)» specifies the latest possible moment the phenomenon could have ended to exist.

Use this option, if the phenomenon did surely not existing after the date you enter and may have existed before and during this date.
